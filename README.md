# dotfiles-popos
Pop!_OS用のDotfilesです

# Before Git Clone

actions below cannnot be done by this repository's scripts.

## apt update & upgrade
```
sudo apt update
sudo apt upgrade
```

## git install
`sudo apt install git`

## ssh-keygen
```ssh-keygen```

then, `cat ~/.ssh/id_rsa.pub` and copy&paste your public key of your rsa into github's setting.

after that, `ssh -T git@github.com` to confirm your ssh key is ready.
